# 🌱 Estufa Dashboard IoT

Sistema web desenvolvido em **React** para monitoramento e controle de estufas agrícolas (Alface e Repolho), baseado em um projeto de automação IoT com ESP32.

---

## 📌 Sobre o Projeto

Este sistema simula um dashboard de monitoramento agrícola, permitindo visualizar:

- 🌡 Temperatura
- 💧 Umidade do ar
- 🌱 Umidade do solo
- ☀ Intensidade luminosa
- 🚿 Status da irrigação
- ⚠ Alertas do sistema
- 📊 Histórico de dados

---

## 🧠 Contexto

O projeto foi inspirado em um sistema de automação agrícola com IoT, onde sensores e atuadores controlam estufas automaticamente, trazendo benefícios como:

- Redução de desperdício de água  
- Menor erro humano  
- Melhor qualidade das hortaliças  
- Monitoramento em tempo real  

---
